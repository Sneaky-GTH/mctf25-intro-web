Evidently we're going to need a different method (pun intended) to get the data. Thankfully, Firefox has a very powerful tool to do exactly that - Inspect.

To open Inspect, there are several options - use the shortcut "Ctrl+Shift+C", right click and press "Q" or right click and select "Inspect"

Now be not afraid of all the sudden boxes and text that appeared on the bottom of the screen! The part we are most interested in is the Network tab. Here you can see all the requests that your browser is making! You might notice one request being sent every few seconds which is "progress" - you can ignore that, that's things the server is requesting for this website to work.

We're interested in a different type of request - remember how when clicking the button, our browser executed fetch() to try and get data from the server? Every request made by the browser, including fetch(), appears in the Network tab!

Try clicking the button and pay close attention to any requests that appear in the Network tab.

Click the button to proceed! (due to how I made this website, you need to close the pop-up to press the button. Oops!)
